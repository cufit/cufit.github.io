안녕하세요 오픈라이더입니다.

오픈라이더 R1센서을 구매해주셔서 진심으로 감사드립니다.


1. 센서 모드 확인 방법

 1) 현재 판매되는 센서는 최초에 스피드모드로 설정되어 발송해드리고 있습니다. 

 2) 센서 모드를 확인하기 위해서, 센서 배터리를 뺏다가 다시 끼우시면 잠깐 동안 센서 앞쪽에 빨간불 혹은 파란불이 표시됩니다.

 3) 빨간불인 경우 속도계 모드이고, 파란불인 경우는 케이던스 모드인 상태입니다.


2. 센서 연결 방법

 1) 오픈라이더 App에서 '설정 > 센서'메뉴 선택합니다.

 2) 속도계 센서(혹은 케이던스 센서) 메뉴를 선택하여 센서를 검색합니다. 이때 메뉴는 센서 모드에 맞게 선택하여야 합니다.

 3) 속도계 센서(혹은 케이던스 센서)가 정상적으로 연동되었으면 센서이름과 버전정보가 표시됩니다.

  - 속도계 모드인 경우, 'OR100-003280 - V3.5 - 100%' 형식으로 표시됩니다.

  - 케이던스 모드인 경우, 'OR200-003280 - V3.5 - 100%' 형식으로 표시됩니다.

 4) 정상적으로 센서 데이터 수신여부되는지 확인하시려면, 현재 화면에서 센서를 몇 십 초정도 돌려주시면 아래와 같이 표시되는 걸 확인하실 수 있습니다.

  - 속도계 모드인 경우, 'OR100-003280 - V3.5 - 100% (7.9Km/h)' 맨 끝에 스피드 데이터가 추가적으로 표시됩니다.

  - 속도계 모드인 경우, 'OR100-003280 - V3.5 - 100% (75rpm)' 맨 끝에 케이던스 데이터가 추가적으로 표시됩니다.

 5) 센서 데이터 수신까지 확인하셨다면, 센서 연결이 완료된 상태입니다.

 

3. 펌웨어 업데이트 방법

 1) 위 '센서 연결 방법'과 센서를 연결합니다. 

 2) 속도계 센서(혹은 케이던스센서) 가 정상적으로 연동되었으며, 현재 센서의 펌웨어 버전에 따라 '업데이트' 버튼이 회색에서 파란색으로 활성화됩니다.

 3) 만일 '업데이트' 버튼이 활성화 되어 있으시면, 센서를 스마트폰 가까이 둔 상태로 '업데이트' 버튼을 선택하여 업데이트를 시작합니다.

 4) 약 1분간 업데이트가 진행되며 업데이트가 완료되면 센서는 스피드모드로 초기화되면 LED는 빨간색으로 표시됩니다. 

 5) 업데이트 완료 후 다시 속도계 센서로 연동 후 속도계로 사용하시거나 아니면 '모드변경' 버튼을 클릭하여 케이던스 모드로 변경하실 수 있습니다.


4. 센서모드 변경 방법 

 1) 위 '센서 연결 방법'과 센서를 연결합니다. 

 2) 센서가 정상적으로 연동되었다면 '모드변경' 버튼이 활성화됩니다.

 3) '모드변경' 버튼을 클릭하면 모드 변경을 확인하는 메세지창이 표시되고, 확인 버튼을 선택하시면 모드변경이 시작됩니다.

 4) 잠시 후 센서 앞쪽에 빨간불 혹은 파란불이 표시되면 모드 변경이 완료된 상태입니다.

 5) 변경된 모드로 다시 센서 연결하여 사용하시면 됩니다. (위 '센서 연결 방법' 참고)
